# GanaKana

It is a toy project.

Learn Japanese Hiragana, Katakana and Youon with WinUI3!

## Features

- Quizzes about the corresponding Romaji
- Choose whether to enable Hiragana/Katakana and Youon!
- Japanese Alphabet for Reference
